Customer Churn Prediction

Project Overview

This project predicts customer churn using Machine Learning techniques. The goal is to identify customers who are likely to leave a telecom service based on customer behavior and subscription details.

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Machine Learning Models

- Logistic Regression

Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Data Preprocessing
5. Model Training
6. Model Evaluation

Key Insights

- Customers with shorter tenure are more likely to churn.
- Long-term contracts reduce churn probability.
- Higher monthly charges are associated with increased churn.

Model Performance

- Logistic Regression Accuracy: 82%
